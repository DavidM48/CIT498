Project Overview

For my project, I want to use machine learning in order to solve a Rubik’s Cube.

This project addresses the need for using machine learning to solve a Rubik’s cube like how humans would.

An objective of this project is to use Q-Learning to solve a Rubik’s cube.

This will be limited to using a Raspberry Pi.
